### Requirement
- `PHP >= 8.1`

### Clone source code
Make a folder on your local and clone source on your local
- ``` git clone```

### Install source
- `cd to root folder`
- `composer i`
- `cp .env.example .env`
- `php artisan key:gen`
- `config info DB,... in the .env file`
- `php artisan migrate`

### Run app
- ``php artisan serve``
- ``or setup vitual host``

### Analyze code
- ``composer analyze PATH_FOLDER_OR_FILE``

### Fix format
- ``composer fix-format PATH_FOLDER_OR_FILE``
